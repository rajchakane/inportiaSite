  • Syllabus

Aim: Learn Selenium test automation from basics to expert level. To build automation framework from scratch. Become an expert in Behavior driven development (BDD) test framework.

Prerequisites: Basic Manual Testing Concepts

Duration: 60 hours practical classroom training for 8 weeks.

Tools/Frameworks/Libraries: Core Java, Selenium WebDriver, Cucumber, Git, Maven, Jenkins

Result: Candidates will understand different entities involved in automation framework. Can make small POC's(proof of concept) related to automation. Crack automation interviews related to Core Java, Selenium, BDD.

Audience: Anyone who wants to make career in IT automation.

Project Covered: Automation Framework which will handle e2e scenarios of a web application.
